# sudo ku Privacy Policy

## 1. Introduction
When you use our website and the game services we provide, you trust us with your information.  
We understand that this is a significant responsibility, so we work to protect your information
and ensure that you remain in control of it.
  
With this privacy policy, we aim to help you understand which data we collect and why we collect it, and how you can update, manage, and delete your information.
  
Within our services, you can use features such as solving sudoku offline, organizing online sudoku competitions with your friends, and viewing your game statistics.
  
## 2. Information We Collect
The categories of the information we collect are listed as follows:
### Account Data:
- User Name
- Mail Address
- Password
- First Name
- Second Name
- Avatar Image
  
### Game Data:
- Multiplayer Match Participation
- Current Game State (Sudoku Board State)
- Match Status
  
### Statistics Data:
- Number of Correct Moves in Multiplayer Mode
- Win/Loss Result in Multiplayer Mode
- Game Completion Time
- Number of Match
  
### Technical Data:
- IP Address (for connection management and security purposes)
  
## 3. Data Retention Period
Data is retained for as long as necessary for the services to function.
When a user account is deleted, the corresponding data is removed from the system.
  
## 4. How We Use Information
The collected data is used to enable the games to be played, allow users to personalize their accounts, and track their own progress:
- Manage User Accounts
- Display Users' Statistics
- Match Players
  
## 5. Data Sharing
User data is not sold to other companies, is not shared for commercial purposes, 
and is not shared with third parties except as required by law.
  
## 6. Data Security
We take necessary technical measures to ensure the security of user information:
- Protection of Data Against Unauthorized Access
- System Security

### Efforts to Ensure Data Security
- Passwords are not stored in plain text and are stored using secure hash algorithms.
- Email and password verification are enforced using secure methods to ensure account protection and prevent unauthorized access.
  
## 7. User Rights
Users may have certain rights over their data:
- View Account Information
- Update Information
- Request Account Deletion
  
## 8. Changes to This Policy
This privacy policy may be updated from time to time. Changes will be published on this page.
