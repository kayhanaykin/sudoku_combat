# django: The core web framework itself.

# djangorestframework & djangorestframework-simplejwt: 
#  These are extensions for Django to easily build REST APIs 
#  and handle JWT (JSON Web Token) authentication for your users.
#  REST stands for REpresentational State Transfer.
#  JSON (JavaScript Object Notation). XML, YAML, CSV, HTML, Binary and etc.

# django-cors-headers:
#  due to microservice structure, react request from a different port 
#  browers have same-origin Policy, and blocks requests to other ports
#  this header allows requests from other ports by adding Access-Control-Allow-Origin header to responses

# requests:
#  to make HTTP request from backend
#  required for remote auth.
#  send GET req to 42 intra

# daphne & channels: add WebSocket support to Django
#  Standard Django uses a "WSGI" server, only HTTP reqs.
#  daphne is the ASGI (Asynchronous Server Gateway Interface) server and takes over
#  channels is a framework that allows adding WebSocket support to Django
#  channels-redis is a Redis backend for channels
#  Redis is like db but on ram and fast
